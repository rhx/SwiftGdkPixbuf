PixbufSimpleAnimClassProtocol

public protocol PixbufSimpleAnimClassProtocol

The PixbufSimpleAnimClassProtocol protocol exposes the methods and properties of an underlying GdkPixbufSimpleAnimClass instance. The default implementation of these can be found in the protocol extension below. For a concrete class that implements these methods and properties, see PixbufSimpleAnimClass. Alternatively, use PixbufSimpleAnimClassRef as a lighweight, unowned reference if you already have an instance you just want to use.
